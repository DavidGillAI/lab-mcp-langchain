This lab demonstrated how to integrate MCP (Model Context Protocol) with LangChain. A custom MCP math server was created exposing both a tool (`add`) and a resource containing server information. Using `MultiServerMCPClient`, MCP tools and resources were loaded into LangChain, and an agent was created with `ChatOpenAI` and `create_agent`. The agent successfully used the MCP tool to perform calculations and used MCP resource content as context when answering questions. Compared to direct API integration, MCP provides a standardized and reusable interface for connecting external systems and tools, making it easier to add new integrations and reuse them across frameworks. Direct API integration offers more control and potentially better performance, but requires custom code and maintenance for each integration. During development, MCP tool loading in Jupyter Notebook encountered a Windows/IPyKernel subprocess issue (`UnsupportedOperation: fileno`), so the final implementation was demonstrated using standalone Python scripts.
